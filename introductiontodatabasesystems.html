    what is database systems?

    A database is an organized collection of data, generally stored and accessed electronically from a computer system. Where databases are more complex they are often developed using formal design and modeling techniques. -A Database Management System (DBMS) is a system (software) that provides an interface to database for information storage and retrieval.
